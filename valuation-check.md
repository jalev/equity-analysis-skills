> **Format (follow AGENTS.md rules):** Write narrative paragraphs — not bullet lists. Use tables for scenario ranges and multiples comparisons. Cite as `[FY20XX 10-K, Item 6]` — never line numbers. The `→` after a quote must add analytical interpretation, not paraphrase the quote.

Provide a valuation assessment using available financial data. This is a
sanity check, not a precise model.

Cover:

1. **Relative valuation** - current P/E, EV/EBITDA, P/FCF vs:
   - Own 5-year historical average
   - 3-5 direct peers (name them)
   - Sector median
2. **Growth-adjusted view** - PEG ratio, EV/EBITDA-to-growth
3. **Simple intrinsic value estimate** - reverse DCF: what growth rate is
   the current price implying? Is that reasonable given historical performance?
4. **Margin of safety** - at current price, what needs to go right?
   What is priced in vs what is not?
5. **Scenario framing**:
   - Bull case (top-quartile outcome): fair value range
   - Base case (continuation of trends): fair value range
   - Bear case (structural deterioration): fair value range

State all assumptions explicitly. Flag where data is insufficient.
